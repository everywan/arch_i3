== 介绍
各部分介绍如下
* install: 安装相关
    ** install/script 软件的安装脚本
    ** install/script/gui 当作为主力电脑, 日常使用时, 软件的安装脚本.
    ** install/script/server 当作为服务器时, 软件的安装脚本.
    ** install/docker 各种软件的docker版本, 如 hive,spark等, 根据需要安装.
* faq.md: 安装/使用上可能需要的问题解答.
